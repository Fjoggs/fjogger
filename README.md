# fjogger
Barebones 'log-to-file' package for Javascript

## Usage

```js
import log from 'fjogger'

log('error', 'something went wrong');
```

logs to `/logs/log.txt`

## Frameworks used
- Ecmascript 6
- Jest

## Dev related frameworks
- Eslint with prettier config
- Babel
